Lorentz Light-Cone Model (LLCM)

Interactive paper page for GitHub Pages deployment. Expand sections below to explore the core claims, formulas, evidence, and reproducibility commands.

Core formula

# Standard attention (Euclidean)
score = Q @ K.T / sqrt(d)

# LLCM attention (Lorentzian, formula F3)
score = (-σ * Qt @ Kt.T + Qs @ Ks.T) / sqrt(d)

The minus sign is treated as a geometric consequence (Lorentzian signature), not a tuning convention.

What changes in one line

Attention scoreQ·K (isotropic)−σ·Qt·Kt + Qs·Ks (anisotropic)

Stability boundaryd_c = 0d_c > 0
Why the minus sign is necessary

Under conditions R, E, and T on displacement cost structure, the leading quadratic form is indefinite; in (1+1)D this yields Lorentzian signature as the algebraic outcome.

Quick start (Python) 
from dataclasses import dataclass
import torch
from lorentz_transformer import LorentzMultiHeadAttention, MinkowskiLayerNorm

@dataclass
class Config:
    d_model: int = 256
    n_heads: int = 8
    formula: str = 'f3'
    time_ratio: float = 0.25
    dropout: float = 0.1

config = Config()
attn = LorentzMultiHeadAttention(config)
norm = MinkowskiLayerNorm(config.d_model)

x = torch.randn(2, 16, config.d_model)
out, weights = attn(x)
out = norm(out)
print(f"σ = {attn.sigma:.3f}")
Experimental highlights
